## Development Backend

Open your favorite Terminal and run these commands.
To run code backend :

```sh
go run main.go
```
To add admin :

```sh
localhost:8080/admin/register
```
To login admin:

```sh
localhost:8080/admin/login
```
Json data admin register:
```sh
{  
 "username":"admin",
  "phone":"12345",
  "email":"admin123",
  "password":"admin12345"
}
```
Json data admin login:
```sh
{  
  "email":"admin123",
  "password":"admin12345"
}
```

Client same like admin to register and login
